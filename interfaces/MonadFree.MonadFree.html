Interface MonadFree<F, M>

A structure which able to lift up in F.

An instance of Functor<F> must satisfy following rules:

  • Identity: map(id) == id,
  • Composition: for all f and g; map(pipe(f)(g)) == pipe(map(f))(map(g)).
